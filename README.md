# NightOwl Coffee Co.
Nightowl Coffee Co. E-commerce Application

Nightowl Coffee Co. is a web based e-commerce app built with Ruby on Rails.

Features:
- Welcome page
- List of products
- Single product view
- Add quantity for products
- Add products to cart
- Create new user (sign-up)
- Log in / log out
